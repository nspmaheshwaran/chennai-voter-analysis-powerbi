

# Chennai Voter Insights Dashboard

## 📊 Project Overview
This project analyzes voter distribution across Chennai constituencies using Power BI.

## 🎯 Objective
To understand demographic distribution and constituency-wise voter patterns.

## 📁 Dataset
- Type: Sample/Public dataset
- Tool Used: Excel for preprocessing

## 📌 Key Insights
- Female voters are slightly higher (~51.7%)
- Perambur has the highest voter count
- Harbour has the lowest voter count

## 🛠 Tools Used
- Microsoft Excel
- Power BI
